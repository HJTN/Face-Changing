# Material Design Guidelines

## Overview
Material Design은 Google이 2014년 발표한 디자인 시스템으로 물리 세계의 질감·깊이·움직임을 디지털 인터페이스에 적용한다. Material Design 3(Material You, 2021)는 개인화·다이나믹 컬러를 강조한다.

## Material Design 3 핵심 원칙

### 1. Material as Metaphor (물질 은유)
- 물리적 표면과 빛을 모방 → 그림자·elevation으로 계층 표현
- 표면(Surface)이 기본 빌딩 블록

### 2. Bold, Graphic, Intentional
- 강한 타이포그래피, 의도적 색상 사용
- 화이트스페이스를 활용해 정보 전달

### 3. Motion Provides Meaning
- 모든 애니메이션이 의미를 가져야 함
- Shared Element Transition — 화면 전환 시 요소 연속성

### 4. Adaptive Design
- 폰·태블릿·데스크탑·폴더블 모두 대응
- Dynamic Color — 사용자 배경화면에서 색상 팔레트 추출

## 핵심 컴포넌트 (Material 3)

| 컴포넌트 | 용도 |
|---|---|
| FloatingActionButton | 주요 액션 (화면당 1개) |
| NavigationBar | 하단 탭 네비게이션 |
| TopAppBar | 상단 제목·액션 |
| Card | 콘텐츠 그룹화 |
| Chip | 필터·태그 |
