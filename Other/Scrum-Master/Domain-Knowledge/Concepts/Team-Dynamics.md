# Team Dynamics

## Overview
팀 역학(Team Dynamics)은 팀 구성원 간의 상호작용·관계·역할이 팀 성과에 미치는 영향을 이해하는 것이다. 스크럼 마스터는 팀 역학을 이해하고 건강한 팀 환경을 조성한다.

## 터크만 모델 (Tuckman's Stages)

`
Forming (형성) → Storming (폭풍) → Norming (규범화) → Performing (성과) → Adjourning (해산)
`

| 단계 | 팀 특성 | 스크럼 마스터 역할 |
|---|---|---|
| **Forming** | 조심스러움, 서로 파악 중 | 구조·기대치 명확화, 안전 환경 조성 |
| **Storming** | 의견 충돌, 갈등 발생 | 갈등 조정, 팀 규범 수립 지원 |
| **Norming** | 팀 규범 정착, 협업 시작 | 자율성 부여, 프로세스 최적화 |
| **Performing** | 높은 성과, 자기 조직화 | 장애물 제거, 팀 가속화 |
| **Adjourning** | 프로젝트 종료, 해산 | 학습 회고, 성과 축하 |

## 심리적 안전감 (Psychological Safety)

Google Project Aristotle 연구 결과: 팀 효과성의 가장 중요한 요인은 **심리적 안전감**이다.

### 심리적 안전감이 있는 팀의 특징
- 의견 충돌·실수를 두려움 없이 이야기할 수 있음
- 비판 없이 이상한 아이디어를 꺼낼 수 있음
- "모른다"고 솔직하게 말할 수 있음

### 스크럼 마스터가 안전감을 조성하는 방법
- Blameless Retrospective — 개인이 아닌 시스템·프로세스에 집중
- 리더가 먼저 취약함 보이기 ("나도 이것을 잘 몰랐습니다")
- 다양한 의견을 적극적으로 끌어내기
