# The 12 Principles of Animation

## Overview
Disney 애니메이터 Ollie Johnston과 Frank Thomas가 1981년 저서 「The Illusion of Life」에서 정리한 12가지 애니메이션 원칙이다. 원래 전통 애니메이션을 위한 원칙이지만, UI 모션 디자인에서도 핵심 기준으로 활용된다.

## 12가지 원칙

| # | 원칙 | UI 적용 |
|---|---|---|
| 1 | **Squash & Stretch** | 버튼 눌림 시 살짝 찌그러졌다가 튀어오름 |
| 2 | **Anticipation** | 드롭다운 열리기 전 살짝 수축 |
| 3 | **Staging** | 중요한 UI 요소는 여백으로 강조 |
| 4 | **Straight Ahead & Pose to Pose** | 키프레임 기반 UI 전환 |
| 5 | **Follow Through & Overlapping Action** | 카드 이동 후 그림자가 살짝 늦게 따라옴 |
| 6 | **Slow In & Slow Out** | Ease-in-out 이징 커브 |
| 7 | **Arc** | 요소가 직선이 아닌 자연스러운 호를 그리며 이동 |
| 8 | **Secondary Action** | 알림 등장 시 배경 살짝 어두워짐 |
| 9 | **Timing** | 빠른 것=가벼움, 느린 것=무거움 느낌 부여 |
| 10 | **Exaggeration** | 삭제 확인 다이얼로그의 흔들림 강조 |
| 11 | **Solid Drawing** | 3D감과 무게감 표현 |
| 12 | **Appeal** | 전체적으로 매력적이고 개성 있는 모션 |

## UI 모션 설계 시 우선 적용 원칙
1. **Slow In & Slow Out** (6) — 거의 모든 UI 애니메이션에 필수
2. **Timing** (9) — 속도가 사용자 인식 결정
3. **Anticipation** (2) — 다음 동작 예측 가능하게
4. **Follow Through** (5) — 자연스러운 관성 표현
