# Animation Principles in Motion Design

## Overview
모션 디자인에서 애니메이션 원칙은 움직임이 자연스럽고 의도적으로 보이게 하는 기본 규칙이다. Disney의 12 Principles를 디지털 인터페이스에 적용하여 사용자 경험을 향상시킨다.

## UI 모션의 4가지 목적 (Material Design 기준)

| 목적 | 설명 | 예시 |
|---|---|---|
| **Informative** | 공간·계층 관계 전달 | 화면 전환, 드롭다운 |
| **Focused** | 주의 집중 유도 | 에러 흔들기, 알림 뱃지 |
| **Expressive** | 브랜드 개성 표현 | 온보딩 일러스트 애니메이션 |
| **Delightful** | 사용 즐거움 제공 | 좋아요 폭죽 효과 |

## 핵심 모션 개념

### Easing (이징)
- **Linear** — 일정한 속도. 기계적이고 부자연스럽게 느껴짐
- **Ease-in** — 천천히 시작해서 빠르게 끝남. 요소가 사라질 때
- **Ease-out** — 빠르게 시작해서 천천히 끝남. 요소가 나타날 때
- **Ease-in-out** — 부드럽게 시작하고 끝남. 이동 애니메이션

### Timing (타이밍)
- 마이크로인터랙션: **100–300ms** (빠른 피드백)
- 화면 전환: **200–500ms**
- 복잡한 애니메이션: **500ms–1s**
- 1초 이상 → 사용자가 기다린다고 느낌 (로딩 스피너 필요)
