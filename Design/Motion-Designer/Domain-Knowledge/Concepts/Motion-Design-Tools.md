# Motion Design Tools

## Overview
모션 디자이너가 사용하는 도구는 프로토타입(빠른 개념 검증)부터 최종 납품(After Effects 파일, Lottie JSON)까지 목적에 따라 다르다.

## 도구 생태계

### 프로토타이핑
- **Figma (Smart Animate)** — UI 컴포넌트 간 전환 프로토타입
- **Principle** — Mac 전용, 복잡한 인터랙션 프로토타입
- **ProtoPie** — 코드 없이 고급 인터랙션 구현

### 전문 모션 도구
- **After Effects** — 업계 표준. 복잡한 애니메이션, 영상 편집
- **Cinema 4D / Blender** — 3D 모션 그래픽
- **Rive** — 인터랙티브 애니메이션. 게임·앱에 임베드 가능

### Web/App 납품
- **Lottie (AE → JSON)** — After Effects 애니메이션을 JSON으로 변환, 앱/웹에서 재생
- **CSS Animation** — 간단한 UI 애니메이션
- **GreenSock (GSAP)** — 고성능 JavaScript 애니메이션 라이브러리

## 파일 포맷

| 포맷 | 용도 | 크기 |
|---|---|---|
| **Lottie JSON** | 앱·웹 인터랙티브 애니메이션 | 매우 작음 |
| **WebM/MP4** | 영상 자동재생 | 중간 |
| **GIF** | 이메일·단순 반복 | 큼 (최적화 필요) |
| **APNG** | PNG 투명 배경 애니메이션 | 중간 |
