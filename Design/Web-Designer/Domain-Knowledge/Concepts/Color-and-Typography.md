# Color and Typography

## Overview
색상과 타이포그래피는 웹 디자인의 두 핵심 시각 언어다. 올바른 선택은 브랜드 정체성을 확립하고 가독성·접근성·사용자 경험을 결정한다.

## 색상 (Color)

### 색상 팔레트 구성
- **Primary** — 브랜드 핵심 색상 (CTA, 로고, 강조 요소)
- **Secondary** — 보조 색상 (배경, 구분선)
- **Neutral** — 텍스트·배경에 사용하는 무채색 계열
- **Semantic** — 기능적 색상: 성공(초록), 경고(노랑), 오류(빨강), 정보(파랑)

### 접근성 기준 (WCAG 2.1)
- 일반 텍스트: 명도 대비 **4.5:1** 이상
- 큰 텍스트(18pt+): **3:1** 이상
- UI 컴포넌트: **3:1** 이상
- 도구: Colour Contrast Analyser, Figma 플러그인 A11y

## 타이포그래피 (Typography)

### 폰트 스케일 (Type Scale)
`
Display: 48–96px  — 히어로 헤드라인
H1:      36–48px  — 페이지 제목
H2:      28–36px  — 섹션 제목
H3:      22–28px  — 서브섹션
Body:    16–18px  — 본문 (최소 16px 권장)
Caption: 12–14px  — 캡션·레이블
`

### 가독성 원칙
- **Line-height**: 본문 1.5–1.8배, 제목 1.1–1.3배
- **Line-length**: 한 줄 45–75자 (약 600px 컨테이너 기준)
- **Letter-spacing**: 대문자·캡션은 0.05em 이상 트래킹 추가
