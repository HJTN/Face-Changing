# Visual Hierarchy

## Overview
시각적 위계(Visual Hierarchy)는 디자인 요소의 중요도를 시각적으로 표현해 사용자가 무엇을 먼저 볼지, 어디에 주의를 집중할지 안내하는 원칙이다.

## 위계를 만드는 요소

| 요소 | 방법 | 예시 |
|---|---|---|
| **크기 (Size)** | 크면 더 중요하게 인식 | H1 > H2 > H3 > Body |
| **색상 (Color)** | 강조색으로 시선 유도 | CTA 버튼은 브랜드 강조색 |
| **대비 (Contrast)** | 높은 대비 = 더 눈에 띔 | 검정 텍스트 on 흰 배경 |
| **위치 (Position)** | 좌상단 → 우하단 순서로 읽음 (F/Z 패턴) | 로고는 항상 좌상단 |
| **굵기 (Weight)** | Bold 텍스트가 먼저 읽힘 | 제목은 Bold, 본문은 Regular |
| **여백 (Spacing)** | 여백이 많을수록 중요도 높아 보임 | 히어로 배너의 여백 |
| **순서 (Sequence)** | 번호/화살표로 읽는 순서 강제 | 온보딩 스텝 번호 |

## F-패턴 vs Z-패턴
- **F-패턴**: 텍스트 많은 페이지에서 나타남 (블로그, 뉴스). 첫 두 줄 → 왼쪽 세로 스캔
- **Z-패턴**: 텍스트 적은 페이지에서 나타남 (랜딩 페이지). 좌상 → 우상 → 좌하 → 우하

`
