# Visual Design Fundamentals

## Overview
시각 디자인의 기본 원칙은 정보를 명확하고 매력적으로 전달하기 위해 시각적 요소를 배치·조합하는 방법을 다룬다. UI 디자이너가 아름다운 것과 기능적인 것을 동시에 달성하기 위한 토대다.

## 핵심 원칙

### 1. Visual Hierarchy (시각적 위계)
중요도에 따라 요소의 크기·색상·위치·굵기를 차별화해 시선을 유도한다.

### 2. Contrast (대비)
명도·색상·크기 대비로 가독성과 집중도를 높인다. WCAG 기준: 텍스트 명도 대비 4.5:1 이상.

### 3. Alignment (정렬)
요소들을 보이지 않는 격자선에 정렬해 질서와 전문성을 표현한다.

### 4. Proximity (근접성)
관련 있는 요소는 가까이, 관련 없는 요소는 멀리 배치해 그룹을 암시한다.

### 5. Repetition (반복)
색상·폰트·아이콘 스타일을 반복해 일관성과 브랜드 정체성을 구축한다.

### 6. White Space (여백)
의도적인 빈 공간으로 가독성·집중도·고급감을 높인다.

## 색상 이론 요약
- **색상환** — Hue(색조), Saturation(채도), Value(명도)
- **보색 조합** — 강한 대비로 주목도 향상
- **유사색 조합** — 조화롭고 차분한 느낌
