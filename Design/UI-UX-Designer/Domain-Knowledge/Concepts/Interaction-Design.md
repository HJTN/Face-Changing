# Interaction Design

## Overview
인터랙션 디자인(IxD)은 사용자와 제품·시스템 사이의 상호작용 방식을 설계하는 분야다. 클릭·스와이프·입력 등 모든 사용자 행동에 대한 시스템의 반응을 정의한다.

## 5가지 핵심 차원 (Gillian Crampton Smith)

| 차원 | 설명 |
|---|---|
| **Words** | 버튼 레이블·안내문 등 텍스트 |
| **Visual Representations** | 아이콘·이미지·타이포그래피 |
| **Physical Objects/Space** | 기기 형태·입력 방식 (터치, 키보드) |
| **Time** | 애니메이션·사운드·피드백 타이밍 |
| **Behaviour** | 규칙·반응·오류 처리 방식 |

## 핵심 개념

- **Feedback** — 사용자 행동에 즉각적인 반응 제공 (버튼 눌림, 로딩 스피너)
- **Affordance** — 요소 자체가 사용 방법을 암시 (버튼은 눌릴 것 같이 생겨야 함)
- **Mental Model** — 사용자가 시스템이 어떻게 동작할 것이라 기대하는 모델
- **Error Prevention** — 실수가 발생하기 전에 방지하는 설계
- **Microinteraction** — 단일 태스크를 위한 소규모 인터랙션 (좋아요 버튼, 슬라이더)
