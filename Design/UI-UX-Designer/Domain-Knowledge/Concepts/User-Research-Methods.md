# User Research Methods

## Overview
사용자 조사(User Research)는 실제 사용자의 행동·니즈·동기를 이해하기 위해 체계적으로 데이터를 수집하는 활동이다. 디자인 결정의 근거를 주관적 추측이 아닌 증거에 기반하게 만든다.

## 주요 방법론

| 방법 | 유형 | 사용 시점 |
|---|---|---|
| **심층 인터뷰** | 정성 | 사용자 동기·맥락 탐색 |
| **설문조사** | 정량 | 대규모 패턴 검증 |
| **사용성 테스트** | 정성/정량 | 프로토타입 검증 |
| **카드 소팅** | 정성 | 정보 구조 설계 |
| **A/B 테스트** | 정량 | 두 디자인 안 비교 |
| **히트맵/세션 레코딩** | 정량 | 실사용 행동 분석 |
| **Contextual Inquiry** | 정성 | 실제 환경에서 사용자 관찰 |

## 조사 프로세스
1. 리서치 목표 정의 (무엇을 알고 싶은가?)
2. 방법 선택 (정성 vs 정량, 예산·일정 고려)
3. 참가자 모집 (타겟 페르소나 기준)
4. 데이터 수집
5. 분석 및 인사이트 도출
6. 디자인에 반영
